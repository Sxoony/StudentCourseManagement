# StudentCourseManagement


C# Implementation of a program that can perform CRUD on Students and their courses, as well as conjoining both together in a relationship.

## Description
Menu style program that gives the user the capability to add students, have their studentID generated, enroll the student into a course, display the enrolled students, and to remove a student from a course.
This program can be used to sort and maange students of an institution.


## Features
- Add a student + generate unique ID
- Remove a student from a course
- Enroll a student into a course
- Add a course
- Display all the students from a specific course



## Technologies
- Language / Framework: C#
- Libraries: no major
